Solar System

Week 3 | Ruby

We were asked to build a model of a solar system to practice our newfound knowledge of classes. The project came in three waves; in the first: build a Planet class; second wave: create new instances of Planet and accompanying instance methods; third wave: create a CLI that allows a user to access data on the existing solar system, or create their own!
